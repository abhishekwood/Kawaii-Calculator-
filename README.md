Welcome to the **Kawaii Calculator**! This is a simple and cute console-based calculator program written in C.

---

## Description

The Kawaii Calculator program lets you perform basic arithmetic operations such as addition, subtraction, division, multiplication, and modulus with a kawaii-themed interface to make calculations fun and adorable.

---

## Features

- Addition of two integers
- Subtraction of two integers
- Division of two integers
- Multiplication of two integers
- Modulus operation between two integers

## How to Run
- Compile the program using a C compiler:
```bash gcc Kawaii-Calculator.c -o kawaii-calculator  ```
- Run the executable:
``` bash ./kawaii-calculator ```
